ABOUT

In cloud systems, misconfigured files are a big problem making them prone to cyber attacks, which results in data breaches, data loss, financial loss, and most of the time; both.

To counter this, a manual search and identify approach is mostly used which is often very time consuming. Cloud Misconfiguration Detection system is an AI(CNNs) powered model which takes in the misconfigured files from the user and returns the areas where the file is misconfigured and also suggests actions to counter that.

The project is currently under development in National University Of Modern Languages Rawalpindi Campus by Umer Ahmed NAseem CS Department.
